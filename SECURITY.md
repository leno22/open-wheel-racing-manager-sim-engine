# Security Policy

## Supported Versions

The Open-Wheel Racing Manager Sim Engine is currently under active development.

At this stage:
- Only the latest version available on the `main` branch is supported.
- No long-term support (LTS) versions are provided.
- Development builds and experimental features may change without notice.

Security fixes will be applied to the current development version when applicable.

## Scope

This security policy applies to:
- Core simulation logic
- Data handling and persistence
- Save/load systems
- Tooling and scripts included in this repository

This policy does **not** cover:
- Forks or unofficial builds
- Third-party tools or mods
- External assets or dependencies not maintained in this repository
